## 💡 功能说明

### 默认配置
```
周末打卡开关：关闭（默认）❌
节假日打卡开关：关闭（默认）❌
```

**效果**：
- ✅ 工作日（周一～周五）自动打卡
- ❌ 周末（周六、周日）自动暂停
- ❌ 法定节假日自动暂停
- ✅ 调休工作日（如2月14日、2月28日等）自动打卡

### 智能识别逻辑

```
判断今天是否打卡：
1. 查询节假日数据库
   ├─ 找到记录
   │   ├─ type=0 (法定节假日) → 根据"节假日打卡"开关决定
   │   ├─ type=1 (调休工作日) → 正常打卡 ✅
   │   └─ type=2 (自定义休息日) → 根据"节假日打卡"开关决定
   │
   └─ 未找到记录 → 查看星期
       ├─ 周六/周日 → 根据"周末打卡"开关决定
       └─ 周一～五 → 正常打卡 ✅
```

## 🎯 特别提醒

### 1. 春节假期超长
2026年春节从2月15日（周日）一直放到2月23日（周一），**整整9天**，是史上最长的春节假期！

### 2. 中秋国庆分开
2026年的中秋节（9月）和国庆节（10月）不在一起，分开放假：
- 中秋节：9月25-27日（3天）
- 国庆节：10月1-7日（7天）

### 3. 调休工作日
记得关注调休工作日，这些日期虽然是周末，但需要上班：
- 2月14日（周六）- 春节调休
- 2月28日（周六）- 春节调休
- 5月9日（周六）- 劳动节调休
- 10月10日（周六）- 国庆节调休

应用会**自动识别**这些调休工作日，无需手动调整！
